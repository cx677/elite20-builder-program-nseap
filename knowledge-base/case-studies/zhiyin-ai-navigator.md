# Case Study：智引 AI 导航导师

## 项目背景

智引 AI 导航导师是一个面向高校师生的 AI+教育项目，用于帮助用户在学习平台中快速定位功能、理解课程内容并降低平台使用门槛。

## 仓库

```text
https://github.com/a976xw7td/zhiyin-ai-navigator
```

## 为什么适合作为案例

- 有真实 GitHub 仓库。
- 有 README 和提交历史。
- 适合验证 GitHub 检查、AI 初评、飞书记录和作品集展示。
- 能体现 AI+教育方向的真实应用价值。

## 可复用经验

- 真实项目比虚拟 demo 更适合 Showcase。
- README 要让评审快速理解项目对象、场景和价值。
- GitHub 仓库可以作为学习证据，不只是代码托管。

## Agent 使用说明

Review Task Agent 可把该案例作为“真实 AI+教育项目”的参考样例，用于判断学生项目是否有明确用户、场景和产物证据。
