# C07 · AI + 数据处理管道

> 难度：L2（进阶） | 类型：构建型 | 预估耗时：4-5天

## 📋 挑战概述

构建一条"AI驱动的数据处理管道"——让AI自动完成数据的采集、清洗、分析和可视化。你将学会如何让AI处理真实世界的数据，从中提取洞察。

> **核心理念**：数据是AI时代的石油。学会让AI处理数据，等于掌握了炼油技术。

### 前置要求
- 完成C02（AI结对编程）
- 基础的Python知识
- 了解CSV、JSON等数据格式

## 🎯 学习目标

完成本挑战后，你将能够：

1. 用AI编写数据采集脚本（爬虫/API/本地文件）
2. 用AI进行数据清洗和预处理
3. 用AI进行数据分析和生成报告
4. 用AI生成数据可视化图表

## 📦 交付物

| 产物 | 说明 | 最低标准 |
|------|------|---------|
| ① 数据管道代码 | 完整的处理流水线 | 能跑通从输入到输出的全流程 |
| ② 分析报告 | 数据分析结果和洞察 | 包含至少3个发现 |
| ③ 可视化作品 | 数据图表 | 至少2张有意义的图表 |
| ④ AAR复盘 | 课后复盘 | 包含管道设计和数据质量的反思 |

## 🧩 挑战步骤

### Step 1: 选题与数据获取（半天）

**选项A：用已有数据**
- 使用课程群的聊天记录（已导出）
- 使用Elite20班级的提交数据
- 使用公开数据集（Kaggle、天池等）

**选项B：自己采集**
- 用AI写一个爬虫抓取网页数据
- 调用开放API获取数据
- 用AI助手收集问卷数据

> "我想分析Elite20班级群2026年4月至6月的聊天记录。数据在[路径]，请帮我看一下数据格式，然后建议可以做什么分析。"

### Step 2: 数据清洗（1天）

让AI帮你分析和清洗数据：

> "请读取这个CSV文件，告诉我：1. 有多少行数据 2. 有哪些列 3. 有没有空值 4. 有没有异常值。然后生成数据清洗脚本。"

**清洗任务示例**：
- 去除重复数据
- 填充或删除空值
- 统一日期格式
- 分类/编码文本数据

### Step 3: 数据分析（1.5天）

让AI帮你做分析：

| 分析类型 | 示例问题 |
|---------|---------|
| **描述性分析** | 数据的基本统计特征是什么？ |
| **趋势分析** | 随时间的变化趋势？ |
| **对比分析** | A组和B组的差异？ |
| **关联分析** | 哪些变量之间存在相关性？ |

> "基于清洗后的数据，请帮我做以下分析：1. 每个月的消息数量趋势 2. 活跃用户排名 3. 消息类型分布。生成分析脚本，输出分析结果。"

### Step 4: 可视化（1天）

让AI生成可视化图表：

> "基于分析结果，生成以下图表：1. 月活跃度折线图 2. 用户发言量柱状图 3. 消息类型饼图。用Python的matplotlib或plotly生成，保存为图片文件。"

**你的任务**：检查图表是否清晰、准确。让AI调整配色、标注、布局。

### Step 5: 撰写报告 + AAR（半天）

把分析结果整合成一份报告：

> "请基于所有的分析结果和图表，生成一份数据分析报告。包含：1. 数据概况 2. 核心发现 3. 可视化图表 4. 结论与建议。用Markdown格式。"

## 📐 评分标准（Rubric）

| 维度 | 通过 (60%) | 良好 (80%) | 优秀 (100%) |
|------|-----------|-----------|-----------|
| **数据流水线** | 能跑通一个步骤 | 从采集到可视化全流程 | 有模块化设计和错误处理 |
| **数据质量** | 数据是原始的 | 做了基本清洗 | 清洗过程有记录和判断 |
| **分析深度** | 有基本统计 | 有趋势/对比分析 | 有洞察和可操作的建议 |
| **可视化质量** | 有图表 | 图表清晰、标注完整 | 图表设计有叙事性 |
| **报告质量** | 完成了报告 | 结构清晰，有观点 | 报告有可信的数据支撑和 actionable 洞察 |

## 💡 提示与常见坑

- ✅ **先用小数据调试**——取100行数据跑通全流程，再用全量数据
- ✅ **让AI解释每一步在做什么**——不理解的数据处理就是黑箱
- 🚫 **不要跳过数据清洗**——"脏数据进，脏数据出"
- 🚫 **不要只做一个图表**——单一视角容易误导
- ✅ **让AI做敏感度分析**——"如果去掉某个异常值，结论会变吗？"
