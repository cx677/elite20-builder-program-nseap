# C10 · AI教育平台重构（Final Challenge）

> 难度：L3（高级） | 类型：产品型 | 预估耗时：3-4周

## 📋 挑战概述

作为Elite20的最终挑战，你将参与重新构建整个AI教育平台。这不是一个"练习"，而是为NSEAP AI Learning Operating System贡献真实代码。你的产出会成为整个课程的基础设施。

> **核心理念**：你不是课程的学习者，而是下一代AI教育系统的建设者。最终成果将成为NSEAP教育架构的Reference Implementation。

### 前置要求
- 完成C09（真实项目交付）
- 具备全栈开发或Agent开发能力
- 理解NSEAP/KSTAR架构

## 🎯 学习目标

完成本挑战后，你将能够：

1. 为大型教育平台贡献可用代码
2. 设计和实现平台的核心模块
3. 将自己的作品整合到平台生态中
4. 编写可供他人使用的文档和配置

## 📦 交付物

| 产物 | 说明 | 最低标准 |
|------|------|---------|
| ① 平台模块代码 | 你负责的模块的完整代码 | 可集成到主平台 |
| ② 集成文档 | 你的模块如何与其他模块交互 | 其他开发者能理解 |
| ③ 部署配置 | 部署所需的配置文件 | 能一键部署 |
| ④ 演示视频（3分钟） | 展示你的模块在平台中的运行 | 展示端到端功能 |
| ⑤ AAR复盘 | 课后复盘 | 包含对整个项目的反思 |

## 通用工作流

所有方向的Builder按统一流程工作：

```
发现问题 → 提出方案 → AI辅助开发 → 
GitHub提交 → Peer Review → Agent Review → 
Merge → Documentation → Release
```

**特殊要求**：
- C4/C5必须有独立作品
- C7必须发布到公共平台（抖音/小红书/B站/知乎等）
- 所有提交必须有完整的数字留痕（GitHub + 文档）

## 📐 评分标准（Rubric）

| 维度 | 通过 (60%) | 良好 (80%) | 优秀 (100%) |
|------|-----------|-----------|-----------|
| **独立作品** | 有可展示的作品 | 作品可被他人使用 | 作品有用户/被引用 |
| **平台集成** | 本地可运行 | 已集成到主平台 | 有完整的CI/CD和部署 |
| **文档质量** | 有基本文档 | 其他开发者能理解 | 有README+API文档+使用示例 |
| **公共展示** | 在GitHub上 | 在公共平台发布 | 有实际用户反馈/互动 |
| **迭代深度** | 完成1轮迭代 | 基于反馈做了2轮以上迭代 | 形成了可持续优化的机制 |
