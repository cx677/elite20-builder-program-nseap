关于 Elite20 Builder Program

由 佟博士（Richard Tong）— NEOLAF Inc. 创始人、IEEE P3394 工作组成员发起。目标是将 SIAS University Vibe Coding 课程重构为一套可部署、可复制的 Agent Native Learning Operating System。

本项目是 NSEAP 教育架构的 Reference Implementation（参考实现）。任何学校、企业只需部署这套系统即可运行同样的课程。

协调人: 丽霞 (@Cosnπ) · Platform Team: 子安 (牛保康)
